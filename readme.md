# Custom PKGBUILD Repository

A personal repository hosting custom PKGBUILDs for use as a private AUR alternative.

## About

This repository contains curated PKGBUILD scripts for Arch Linux packages that aren't available in the official repositories or that have been customized for specific use cases.

## Features

- Custom-built packages optimized for personal needs
- Regular updates and maintenance
- Community contributions welcome

## Contributing

Found an issue or have a suggestion? Please feel free to:
- Open an issue to report problems
- Submit pull requests with fixes or improvements
- Share ideas for new packages

## Usage

Clone this repository and use it as a local package source with `makepkg` and `pacman`.

`paru` or some other AUR helper can also be configured to use this repository.
